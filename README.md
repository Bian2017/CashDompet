## cash-domept

涉及技术栈: Vue、Vuex、Vue Router、axios、mint-ui

[蓝湖地址](https://lanhuapp.com/web/#/item/project/board?pid=8fc30d97-a4b0-4f68-a553-b474b2847f2b)

### 一、安装依赖

```
npm install
```

### 二、测试环境

```
npm run serve
```

### 三、打包编译

```
npm run build
```

运行该命令，会生成 dist 目录，在 ECS 服务器上指向 dist/index.html 即可。
